# Bazar Ecommerce Sales Analysis ( Interactive Dashboard Creation Using Power BI )

## Project Objective
   Owner of Bazar Store wants us to help them create a dashboard to track and analyze their online sales across india.

## Project key points
 - Data collection
 - Data cleaning
 - Data modeling
 - Data visualization
 - Dashboard design
 - Sharing insights
